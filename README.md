# Selenium Automation Basics

## Tools Used
- Java  
- Selenium WebDriver  
- TestNG  
- Maven  
- ChromeDriver  

## Project Structure
- `src/main/java` – Automation code / Page Objects  
- `src/test/java` – Test scripts / Test cases  
- `testng.xml` – Test suite configuration  

## How to Run the Project
1. Clone the repository  
2. Open the project in Eclipse / IntelliJ IDEA  
3. Run `testng.xml` to execute the full test suite, or run individual test classes  

## Sample Scenarios Automated
- Launch browser and open website  
- Enter username and password  
- Click login button  
- Validate page title  

## Automation Concepts & Skills Covered
- Selenium WebDriver with TestNG for automation testing  
- Data-driven testing using Apache POI with Excel, JSON, and properties files  
- Element identification using XPath and CSS Selectors  
- Page Object Model (POM) using the `@FindBy` annotation  
- Synchronization using implicit, explicit, and fluent waits  
- Handling browser pop-ups, alerts, frames, and multiple windows  
- Mouse interactions using the Actions class  
- Dropdown handling using the Select class  
- CAPTCHA handling using Tesseract OCR
